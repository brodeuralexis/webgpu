# WebGPU

`webgpu` is a cross-platform graphics API in pure Zig. It runs natively on Vulkan, D3D12, Metal, or WebGPU on
WebAssembly.

This API aims to be as close as possible to the [WebGPU standard](https://gpuweb.github.io/gpuweb/) while allowing
diversions where it makes sense in the context of Zig programming.
